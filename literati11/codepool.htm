"Any fool can write a code that computer can understand. Good programmers write code that humans can understand."

-Martin Fowler

A brain-racking algorithm development competition, the Code pool demands the best of approach and coding skills out of you. Write the code as if you know the person who will be maintaining your code is a violent psychopath who knows where you live. Controlling complexity is the essence of computer programming. First, solve the problem. Then, write the code. Only the best will win n ur solution should be best considering time n space complexity.

The event will be conducted online.
First, a problem statement will be released, for which your team needs to develop an algorithm, and code the same in C / C++. Teams will be given a maximum of 3-4 days to solve the same. Your algorithm and code must be efficient in time and space to the extent possible.
Following this, selected teams will have to make a video presentation of their algorithm for the judges where they justify the strength of their algorithm on the basis of the time and space complexities.

Click here to download the problem statement.

Rules & Regulaions:


1. Teams of up to two members only. A team can consist of students from different colleges.

2. There is no requirement for pre-registration.

3. There is no entry fee for this event.

4. Judges’ decision will be final and binding.

5. The coded solution must be submitted along with a file describing your approach.

6. Selected teams would be required to make a video presentation of their solutions.

7. Teams could be required to video-conference for judges' queries.
